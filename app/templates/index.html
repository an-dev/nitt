Nitter is an alternative frontend for Twitter. There used to be many public nitter instances with varying levels of uptime and rate limited, but no more.

This site redirects you to a public instance that is known to be running and not rate limited, all the while distributing the load more evenly across instances so they do not get rate limited as much.

This is done by checking the health of public instances periodically and determining which instances are actually working internally.

You can append any valid Twitter/Nitter URL to this page to be redirected.
For example, click below to be redirected to elon's profile on a working nitter instance:

{{BASE_URL}}/elonmusk
